# POKE BINDER

Pokemon TCG collection tracker with a unified multilingual catalog.

## Import a public pkmn.gg collection

Open POKE BINDER, click `Import`, enter the username from a public pkmn.gg profile,
and click `Import pkmn.gg`. Imported quantities are combined with the local
collection, and equivalent printings are treated as the same card. Re-importing replaces
the previous pkmn.gg contribution without duplicating manually entered cards.
