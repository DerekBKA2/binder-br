# Binder BR

Pokemon TCG collection tracker for Portuguese, English, and Japanese cards.

## Import a public pkmn.gg collection

Open Binder BR, click `Import`, enter the username from a public pkmn.gg profile,
and click `Import pkmn.gg`. Imported quantities are combined with the local
collection and shared between equivalent card languages. Re-importing replaces
the previous pkmn.gg contribution without duplicating manually entered cards.
